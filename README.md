# UpcomingMCU API

The UpcomingMCU API (UMCU API) is a free and open-source REST API built with [Ktor](https://ktor.io/) that consolidates
data about all the Marvel Cinematic Universe (MCU) movies and television shows.

This API allows you to easily discover which MCU production is the next to release and exactly how many days until it
does.
